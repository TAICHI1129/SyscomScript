# SyscomScript Design

## Purpose
A simple, structured scripting language focused on file control and system-like operations.

## Core Principles
- Clear syntax
- Deterministic execution
- Separation between parser and runtime

## Architecture
- Rust: core execution engine
- Python: IDE / REPL only
- VSCode: editor support

## Example Syntax
```
set x = 10
print x
```
